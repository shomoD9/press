# Press V1.5 Private Obsidian Plugin

This plugin is the Obsidian-first shell for Press. After enabling it, use command palette actions that start with `Press:` to install/repair, connect services, run ready check, generate diagrams/plans, build article draft packages, update, and rollback.

## Install (Private Plugin)

1. Copy this folder into your creative vault plugin directory as:
   - `<creative-vault>/.obsidian/plugins/press-v15-private/`
2. In Obsidian, open **Settings -> Community plugins**.
3. Turn off restricted mode if needed.
4. Enable **Press V1.5 (Private)**.
5. Open plugin settings and confirm:
   - Press repository path
   - Creative vault path
   - Excalidraw MCP command

## First Run

1. Run `Press: Install or Repair`.
2. Run `Press: Connect Services`.
3. Run `Press: Ready Check`.
4. Open an essay file and run `Press: Generate Diagram (Current File/Selection)`.
